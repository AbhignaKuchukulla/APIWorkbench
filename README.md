# APIWorkbench

APIWorkbench is a lightweight and developer-friendly tool for testing and managing APIs.  
It provides a streamlined interface to send HTTP requests, save collections, and view responses in real time.

## Features
- Supports common HTTP methods: GET, POST, PUT, DELETE  
- Allows adding custom headers and request bodies  
- Request collections for organizing frequently used APIs  
- Response viewer with JSON formatting and status code display  
- Environment variables for different setups (dev, test, prod)  
- Request history to revisit previous calls  

## Tech Stack
- Frontend: React.js, Tailwind CSS  
- Backend (optional): Node.js, Express  
- Database (optional): MongoDB
